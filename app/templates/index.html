How It Works

An agentic pipeline with end-to-end observability.

Vision + Reasoning

AWS Bedrock multimodal model inspects the damage photo and classifies type, severity, and estimated repair cost.

Tool-Driven Decisions

The agent calls policy tools (inspect, assess, report). Decisions are deterministic and auditable, not a black box.

Full Observability

Every workflow, task, tool, and LLM call is a span in Datadog LLM Observability, with error tracking and SLOs.

Resilient by Design

Automatic retries and model fallback keep the service available when a model endpoint degrades.

Cloud Native

Containerized with Docker and deployed on AWS EC2 for a reproducible, portable runtime.

Auditable Reports

Each decision is persisted as a claim report for compliance and downstream review.
